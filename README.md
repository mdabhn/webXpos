# Simple POS System Interface

This demo project is mainly build for webXgloabl.

## Technology used -

1. ReactJS

2. Bootstrap5 (cdn)

3. TailwindCss

## To run the project follow the steps

1. git clone https://github.com/mdabhn/webXpos.git

2. cd webXpos

3. npm i

4. npm start
